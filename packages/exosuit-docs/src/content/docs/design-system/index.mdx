## Pattern rules

- Use page chrome, dark orientation bands, compact rails, and thin dividers for major visual moments.
- Show what is active, what changed, and what needs attention before adding decorative polish.
- Use semantic color roles, but never rely on color alone.
- Keep Starlight native; the brand follows the host.
