---
title: Exosuit in one page
description: Exosuit is a local-first collaborative cockpit for human and AI development work.
---

Exosuit keeps you and your AI agents working from the same picture of a
project: what the plan is, what's been done, and what's next. You make the
calls; the agent does the work; the workspace holds the shared context so
neither of you loses the thread.

It's built on four design axioms (RFC 10153): inverted sovereignty, high
coherence, phased rigor, and evolutionary intelligence. The day-to-day rhythm
is the SOAR loop: **Status → Orient → Act → Review** (RFC 00224). SOAR is the
loop you run all day. When you sit down to build one discrete thing, you reach
for the heavier PER protocol (Prepare → Execute → Review) instead.
