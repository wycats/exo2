# Security Policy

## Reporting Security Issues

Please report suspected vulnerabilities through GitHub private vulnerability
reporting:

https://github.com/wycats/exo2/security/advisories/new

GitHub private vulnerability reporting must be enabled before this repository
is made public. Avoid posting credentials, exploit details, private project
context, or reproduction steps against a live system in public issues or pull
requests.

## Supported Versions

Exosuit is pre-1.0 software. Security fixes are applied to the active `main`
branch unless a release branch is explicitly announced.

## Public Discussion

General hardening ideas and non-sensitive security improvements are welcome in
public issues and pull requests. Reports that include credentials, exploit
details, private project context, or reproduction steps against a live system
should use the private reporting path.
