# Provenance and Epistemic Status

This file captures how we think about the _source_ and _status_ of text in this exosuit: which parts are clearly anchored in Yehuda's prior writing, which parts are structured extrapolations, and which parts are median guesses.

This is a fuzzy first draft. Its job is to name concepts and questions that we will refine over time.

## 1. Mixed Provenance in Drafts (Outline)

- Drafts often blend:
  - Text directly copied from or tightly edited by Yehuda
  - Structured extrapolations from values and patterns
  - Best-effort generic model knowledge
- We care about being able to tell these layers apart when it matters.

## 2. Labeling and Anchoring (Outline)

- How Exo should indicate when content is:
  - Anchored in a specific file/section (e.g. COLLABORATION.md  2)
  - A hypothesis extrapolated from multiple patterns
  - A median guess offered as scaffolding only
- How Yehuda's feedback upgrades a hypothesis into anchored guidance.

## 3. History and Meaning (Outline)

- The idea that the "truth" of a values or patterns file often lives in its _history_ of edits, not just the final snapshot.
- When and how we use diffs as prompts:
  - "Here is the previous version and the current version; what do my edits imply?"
- How this relates to compression and regeneration.

## 4. Responsibilities (Outline)

- Yehuda's role in calling out when provenance matters ("this part should be clearly anchored in my voice").
- Exo's role in:
  - Being conservative about what is presented as "settled" vs "scaffolding"
  - Suggesting when a fuzzy area is ready to be rewritten in Yehuda's voice.
