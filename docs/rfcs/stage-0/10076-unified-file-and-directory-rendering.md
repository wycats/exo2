# RFC 10076: Unified File and Directory Rendering

## Summary

Unify how the system renders files and directories for humans and tools.

## Motivation

- Consistent UX across CLI and editor.
- Predictable machine-readable representations.

## Proposal

- Define a canonical directory listing schema.
- Define how file references are displayed and serialized.

## Open questions

- Where does pagination belong (CLI, protocol, both)?
- How do we represent symlinks and ignored files?
