# RFC 10123: Ordering Syntax

This RFC has been superseded by RFC 0057.

## Summary

This document originally explored a concrete “ordering syntax” for expressing ordering as data.

## Status

Superseded: ordering is treated as a projection of container shape rather than something encoded as an identity-like syntax.
